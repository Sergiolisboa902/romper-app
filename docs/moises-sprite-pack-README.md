Moises - Sprite Pack (instruções)

Este pacote contém um manifesto e prompts detalhados para gerar sprites do personagem "moises" em estilo 2D Flat Illustration, amigável, premium mobile game, cores vibrantes e sombreamento suave (cel shading). Imagens finais: PNGs 1024x1024, fundo transparente, personagem centralizado.

Conteúdo gerado:
- moises-sprite-pack-manifest.json — lista de frames e nomes de arquivos
- moises-sprite-pack-prompts.md — prompts detalhados em Português e Inglês para cada frame/pose
- moises-sprite-placeholder.svg — placeholder vetorial transparente 1024x1024

Uso rápido:
1. Use os prompts em moises-sprite-pack-prompts.md em um gerador de imagens (Stable Diffusion XL, SD 1.5/2.1 com inpainting, Midjourney, DALL·E) para criar cada PNG.
2. Ajuste seed/variações para consistência entre frames (mesma referência de personagem, paleta e detalhes).
3. Use um artista para retocar e garantir coerência de animação, se necessário.

Se desejar, posso também:
- Gerar variações de prompt em lote
- Criar um spritesheet JSON + layout sugerido
- Ajudar a configurar um workflow com SDXL/InvokeAI/Automatic1111
